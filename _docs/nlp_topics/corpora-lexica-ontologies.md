## Lexical data for historical Dutch
[Data and scripts](https://github.com/cltl/clariah-vocab-conversion) This repository contains several historical and contemporaneous lexicons  for Dutch that have been  converted to one common format (LEMON-RDF). The work has been carried out as part of the  CLARIAH-project "Integrating Diachronous Conceptual Lexicons through Linked Open Data".
