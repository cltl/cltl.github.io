Language of Image and Sound

  • DutchDescriptions: Dutch descriptions for the Flickr30K validation and test data, plus a cross-lingual comparison tool.
  • GroundedTranslation: Multilingual image description
  • Image-Specificity: Reimplementation of Jas & Parikh's (2015) image specificity metric, using word embeddings.
  • SoundBrowser: Interface for the VU Sound Corpus, made in Flask.
  • Spoken-versus-Written: Code and data for our VarDial 2018 paper on spoken versus written image descriptions
  • VU-Sound-Corpus: Collection of crowd-sourced annotations for the Freesound database.
