## VU Reading Machine
The [Reading Machine](https://cltl.github.io/vu-rm-pip3/) provides a language-independent, flexible wrapper together with an up-to-date Dutch NewsReader pipeline. 

## NLPP
The [nlpp repository](https://github.com/cltl/nlpp) provides code to install a bilingual Dutch/English pipeline on Linux.
